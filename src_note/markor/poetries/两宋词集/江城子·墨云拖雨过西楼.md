# 江城子 · 墨云拖雨过西楼

墨云拖雨过西楼。水东流，晚烟收。柳外残阳，回照动帘钩。今夜巫山真个好，花未落，酒新篘(chōu)。
<span class="comment">
巫山：此暗指美人。用巫山神女与楚襄王相会的故事。酒新篘：新漉的酒。，过滤酒。

美人微笑转星眸(móu)。月华羞，捧金瓯(ōu)。歌扇萦(yíng)风，吹散一春愁。试问江南诸伴侣，谁似我，醉扬州。 
<span class="comment">
月华羞：美人笑脸盈盈，顾盼生辉，使姣好的月亮都自愧弗如。歌扇萦风：（美人）翩翩舞扇招来徐徐清风。
