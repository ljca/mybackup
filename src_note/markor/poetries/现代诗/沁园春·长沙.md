# 沁园春 · 长 沙①
<span class="r">毛泽东


<span class="jk">独立寒秋②，湘江北去，桔子洲头。看万山红遍， 层林尽染；漫江碧透，百舸争流③。鹰击长空，鱼 翔浅底，万类霜天竞自由。怅寥廓，问苍茫大地， 谁主沉浮。
<span class="comment">
①此词作于1925年。
②寒秋：深秋。
③舸：大船。
</span>

<span class="jk">携来百侣曾游④，忆往昔峥嵘岁 月稠。恰同学少年，风华正茂；书生意气，挥斥 方遒⑤。指点江山，激扬文字，粪土当年万户侯。 曾记否，到中流击水，浪遏飞舟⑥？
<span class="comment">
④侣：伴侣，此处指同 学。
⑤挥斥：奔放，朝气蓬勃。遒：强劲有力。
⑥遏：阻挡。
</span>
