# 大梦谁先觉

<span class="r">明 · 罗贯

<div class="p">

大梦谁先觉(jué)？平生我自知。
<span class="comment">大梦谁先觉：大梦：道家对人生的一种看法。觉：这里谓觉醒之意。平生：平素，往常。人生只不过是看起来真实一点的梦境，谁能够先看穿这个呢？我生平所为只有我自己知道啊！

草堂春睡足，窗外日迟迟。
<span class="comment">迟迟：徐缓貌。春天我在茅草屋里睡足了觉，窗外的日光慢慢地移动。
