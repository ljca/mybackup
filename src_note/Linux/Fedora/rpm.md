# rpm：Red Hat 系[^redhat]底层软件包管理

## 查询 rpm: `rpm -q`

- `rpm -qf`
- `rpm -qo` 
- `rpm -qi`
- `rpm -ql`

## 安装 `rpm`: `rpm -i `

## 移除/卸载 `rpm`: `rpm -r`

[^redhat]: rpm（Red Hat Package Manager）。所有基于 Red Hat 的 Linux 系统可能使用了不同的软件包管理器解决依赖关系，但在底层都使用了相同的 rpm 软件包格式，Red Hat 系列所包含的 Linux 系统有：Red Hat, Cent OS, Fedora, .....
