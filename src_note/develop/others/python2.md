# Python[^version] 学习笔记

`import` 用来导入 Python 的某个模块即某个库。

而 `from <moudle> import <funcation>` 仅仅用来导入模块中的某个函数。

```python
import math
math.floor(32.9)

from math import floor
print(floor(32.9))
```

> Note！！！

+ Python 语句或者表达式结束不需要使用分号。
+ Python 使用英文字符下的冒号分割语句。
+ Python 变量再声明/定义时不需要使用数据类型。

[^version]: 本篇学习笔记使用的是 `Python2`。
